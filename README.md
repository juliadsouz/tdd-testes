# tdd-testes

📚 Projeto - Git e TDD (Atividade Acadêmica)

Este repositório foi desenvolvido como parte das atividades da disciplina, com o objetivo de praticar o uso de Git, organização de branches e aplicação de testes com TDD.

📂 Organização das Branches

O projeto foi separado da seguinte forma para facilitar a avaliação:

🔹 main

Contém o código desenvolvido durante a aula.
Inclui exemplos, testes e códigos feitos para aprendizado dos conceitos.

🔹 develop

Contém os exercícios de casa solicitados.
Nesta branch foram aplicados, de forma prática, os conceitos aprendidos em aula, especialmente relacionados a TDD.
